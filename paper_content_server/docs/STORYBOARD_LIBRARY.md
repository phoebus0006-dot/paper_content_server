# Storyboard & Film Library

## Image Types

| Type | Description | Display Mode |
|------|-------------|-------------|
| film_still | Real-world film shot | Single frame or analysis card |
| storyboard | Original storyboard drawing | Single frame, comparison, or sequence |
| sequence_frame | Frame within a continuous sequence | 2x2 grid within sequence |

## Learning Priority

Learning value over rigid ratios. The selector prioritizes:

- Excellent real film shots
- Original storyboard drawings
- Storyboard → final shot comparisons
- Shot composition (framing, depth, negative space)
- Character blocking and dialogue coverage
- Lighting and color
- Action continuity and lens transitions

## Display Modes

1. **Single frame** — One image fills the display
2. **Analysis card** — Image with composition overlay and notes
3. **Comparison pair** — Storyboard side-by-side with final shot
4. **2x2 sequence grid** — Four consecutive sequence frames

## Source

Images come from two sources:
1. **Learning Library** — Targeted automatic acquisition of film study materials
2. **Custom Library** — User-uploaded personal study materials

Both libraries are independently managed and share the same safety gates.
